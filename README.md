# Customer Segmentation Analysis - Power BI

## 📊 Overview
This project analyzes customer data to segment customers into High, Mid, and Low value groups based on income and spending behavior.

## 🔍 Key Features
- Dynamic customer segmentation using DAX
- Revenue analysis by segment
- KPI cards for business insights
- Interactive dashboard with slicers

## 📈 Insights
- High value customers contribute most revenue
- Spending behavior varies across income groups
- Opportunity to target mid-value customers

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
